# Порядок работы

1. Загрузить файл, используя кнопку на панели слева или перетащив файл на поле загрузки. Содержимое файла отобразится на странице.
2. Выбрать сеть из выпадающего списка.
3. Если выбрана сеть ДНС, появится дополнительное поле, в котором нужно выбрать формат отчета ("Новый", "Расширенный" или "Старый"). Расширенный подразумевает наличие дополнительных полей с остатками в рублях, себестоимостью и продажами.
4. Выбрать дату отчета.
5. Нажать кнопку "Обработать загруженный файл". На странице отобразится таблца с обработанными данными.
6. Если все верно, можно загрузить файл в формате Excel, нажав на кнопку "Скачать файл".
